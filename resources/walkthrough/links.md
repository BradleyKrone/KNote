# Link your notes

Type `[[` in any note and pick a note title — KNote autocompletes titles,
aliases, `[[Note#headings]]`, and `[[Note#^block-ids]]`.

Ctrl+click a `[[wiki link]]` to follow it (links to notes that don't exist
yet are created on click). Hover a link for a preview.

The **Backlinks** panel (KNote icon in the Activity Bar) shows every note
linking here, plus unlinked mentions you can link with one click. The
**Graph** (`KNote: Open Graph View`) maps the whole vault.
