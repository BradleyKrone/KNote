# Open or create a vault

A KNote **vault** is just a folder of Markdown files.

- Already have one? **File → Open Folder…** — a folder containing `.knote/`
  is picked up automatically.
- Starting fresh? Open any folder and run **KNote: Initialize Vault in This
  Workspace** from the Command Palette (`Ctrl+Shift+P`).

The KNote output channel (View → Output → KNote) shows the vault being
indexed.
